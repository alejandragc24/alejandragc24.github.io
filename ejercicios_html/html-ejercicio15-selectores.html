Tipos de selectores que entran en examen

Clase

Clase(class=""). El .classselector selecciona elementos con un atributo de clase específico. Para seleccionar elementos con una clase específica, escriba un carácter de punto (.), seguido del nombre de la clase. También puede especificar que solo los elementos HTML específicos se vean afectados por una clase. Para hacer esto, comience con el nombre del elemento, luego escriba el carácter de punto (.), seguido del nombre de la clase. Los elementos HTML también pueden hacer referencia a más de una clase.
Ejemplo: .intro { background-color: yellow; }

Identificador

Identificador(id=""). El selector le da estilo al elemento con la identificación especificada.
Ejemplo: #firstname { background-color: yellow; }

Todos los elementos(*)

Todos los elementos(*). El * selector selecciona todos los elementos. El selector * también puede seleccionar todos los elementos dentro de otro elemento.
Ejemplo: * { background-color: yellow; }

Varios elementos

Varios elementos(p em h1 h2).El selector de elementos selecciona todos los elementos con el nombre de elemento especificado.
Ejemplo: p { background-color: yellow; }

Elementos contenidos en otros elementos (ol li)

Elementos contenidos en otros elementos (ol li). El selector element1 ~ element2 coincide con las ocurrencias de element2 que están precedidas por element1. Ambos elementos deben tener el mismo padre, pero element2 no tiene que estar inmediatamente precedido por element1.
Ejemplo: p ~ ul { background: #ff0000; }

Sobrevuleo (p:hover)

Sobrevuleo (p:hover).El :hoverselector se usa para seleccionar elementos cuando pasa el mouse sobre ellos. El selector :hover se puede usar en todos los elementos, no solo en los enlaces.
Ejemplo: a:hover { background-color: yellow; }
